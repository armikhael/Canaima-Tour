Canaima GNU/Linux es uno de los logros más importantes de la revolución Bolivariana y para mi es una honor y un privilegio formar parte de este Proyecto. Aportar mi granito de arena a que esto siga creciendo y con ese fin implante este Paseo para tener un Preview de nuestro sistema operativo. xD

Venezolano y ser Humano como cualquiera.Apasionado por el Desarrollo de Software y Linux. Creyente del Software Libre. Integrante del Equipo de Desarrollo de Canaima GNU/Linux.

Yo me dedico al arte de Canaima como son canaima-bienvenido,Plymouth, tema de iconos entre otros. El estilo de Canaima GNU/Linux esta inspirado en nuestros paisajes más hermosos y esta versión lleva el nombre de kerepakupai o como nosotros lo llamamos el Salto del Ángel.

La accesibilidad es fundamental para la filosofía de Canaima. Creemos que la informática es para todos, independientemente de su nacionalidad, raza, sexo o discapacidad. T_T

Comencé a trabajar como desarrollador de Canaima GNU/Linux en 2009, y ahora se ha convertido en uno de mis pasatiempos más entretenidos. En Canaima decidimos aventurarnos en tratar de combinar la robustez de Gnome con la funcionalidad y vistosidad de Cinammon, para crear algo a lo que llamamos Gnamon. Gnamon es un compendio de temas para GTK2/GTK3, Shell, Metacity, íconos y cursores, con un conjunto de extensiones cuidadosamente seleccionadas. Gnamon está desarrollado para Gnome Shell 3.4, así que si utilizas Debian Wheezy podrás instalarlo. También es el tema que viene por defecto en Canaima 4.0.

Navegador Web Chromium

Acceso a la Internet

Chromium es un proyecto de navegador Web de código abierto que tiene como objetivo construir una manera más segura, más rápida y más estable para todos los usuarios de Internet para disfrutar de la red.

Chromium sirve como base para Google Chrome, El cual fue renombrado a Chromium (nombre y logotipo), con muy pocas adiciones: tales como el seguimiento de uso y auto-actualización de sistema.

Este paquete contiene el Navegador Web Chromium

BEEP

A 2D platformer/shooter game with physics-based puzzles

BEEP is a 2D side-scrolling platformer with physics-based gameplay. You control a small robot equipped with an anti-gravity device, a jet-pack and a gun. Drive, jump, fly, swim and shoot your way through 24 levels scattered across 6 unique environments.

Use BEEP’s anti-gravity device to directly manipulate the physics-based environments. The anti-gravity device is both a tool and a weapon. Use it to smash enemy robots, build towers and solve puzzles.

BEEP has traveled for thousands of years through deep space to explore the Galaxy. Traverse a foreign star system with the BEEP-ship and send robots to the surface of the planets.

Editor de Gráficos Vectorial Inkscape

Creación y edición de imágenes gráficas vectoriales escalables.

Inicia a Inkscape y guarda un subconjunto del formato SVG (Scalable Vector Graphics), un estándar mantenido por el consorcio WWW.

Interfaz de usuario Inkscape le sera sera familiar a CorelDraw o demás programas similares de dibujo. Hay rectángulos, elipses, elementos de texto, imágenes de mapa de bits y curvas a mano alzada. A partir del valor añadido, tanto los objetos vectoriales y de mapa de bits pueden tener transparencia alfa y se pueden transformar arbitrariamente.

Inkscape soporta múltiples archivos abiertos y múltiples vistas por archivo. Los gráficos se pueden imprimir y exportar a mapas de bits PNG.

Algunas de las funciones de importación y exportación se proporcionan con el paquete dia, libwmf-bin, pstoedit, skencil, imagemagick, y perlmagick.

Otras extensiones usan ruby, libxml-xql-perl, python-numpy, y python-lxml. Debe tener estos paquetes para hacer un uso completo de todas las extensiones y efectos.

Si desea utilizar el corrector ortográfico, usted tiene que instalar aspell y language-pack correspondiente, por ejemplo, aspell-es o aspell-de.

Blender

Create and edit 3D models and animations

Blender is an integrated 3d suite for modelling, animation, rendering, post-production, interactive creation and playback (games). Blender has its own particular user interface, which is implemented entirely in OpenGL and designed with speed in mind. Python bindings are available for scripting; import/export features for popular file formats like 3D Studio and Wavefront Obj are implemented as scripts by the community. Stills, animations, models for games or other third party engines and interactive content in the form of a standalone binary and/or a web plug-in are common products of Blender use.

Manipulate the flow of time to solve puzzles Braid is a platform game in painterly style where you manipulate the flow of time to solve puzzles. Every puzzle in Braid is unique; there is no filler.

Braid treats your time and attention as precious, and it does everything it can to give you a mind-expanding experience.

All gameplay is based on time manipulation. Journey into worlds where time behaves strangely; observe, learn from, and then master these worlds.
